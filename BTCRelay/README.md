# BTCRelay

BTCRelay contract is used to store Bitcoin block headers and verify transactions on CyberMiles blockchain.
Only validators on CyberMiles blockchain could submit Bitcoin header to contract, but every users could
use this contract to verify their transaction.

## How to store Bitcoin header

Check [fetchd][] for more details.

[fetchd]: https://github.com/CyberMiles/smart_contracts/tree/master/BTCRelay/fetchd

If fail, `verifyTx` will return 0.
